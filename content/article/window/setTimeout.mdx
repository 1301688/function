- setTimeout()を使用する

- setTimeout とは
  - 指定した時間経過後に処理を実行するときに使用する
  - 指定した時間経過後に処理を実行するメゾット
  - 「タイマー機能」を実装時に使用することが多い
  - clearTimeout()という設定したタイマーを取り消すという機能もある

# ユースケース

## カウントダウンタイマー

```
const countdown = (value, fn, delay = 1000) => {
  fn(value);
  return value > 0
    ? setTimeout(() => countdown(value - 1, fn, delay), delay)
    : value;
};
const log = (value) => console.log(value);
countdown(10, log);
```
