# はじめに、なぜそうなったのか？から学べばキャッチアップしやすい

# 結論

- indexOf()を使用する

- indexOf とは(array)
  - 配列内の要素の添字を求めて、ある要素の存在を求めたり、配列を更新する時に使用する
  - 引数に与えられた内容と同じ内容を持つ最初の配列要素の添字を返すメゾット
  - 配列内の要素をループする時に同時に使用されることが多い
  - 存在しない場合は-1 を返す

# ユースケース

- ある要素の存在をすべて見つける

```
let indices = [];
const arr = ["a", "b", "a", "c", "a", "d"];
const element = "a";
let idx = arr.indexOf(element);
while (idx != -1) {
  indices.push(idx);
  idx = arr.indexOf(element, idx + 1);
}
```
